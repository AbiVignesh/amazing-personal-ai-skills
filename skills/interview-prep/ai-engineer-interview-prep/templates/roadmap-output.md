## Contingency Plan

**If you fall behind:**
- Cut Project 3
- Reduce research depth, focus on systems design
- Shorten some weeks, extend others

**If you're ahead:**
- Add more projects
- Deepen specific areas
- Apply earlier

---

## Red Flags (When to Adjust)

If any of these happen, **stop and recalibrate:**
- [ ] Mock scores not improving week-over-week
- [ ] Shipping a project is taking >6 weeks
- [ ] You can't explain a concept after 2 weeks of study
- [ ] Communication hasn't improved after feedback

**When this happens:** Rerun assessment, adjust plan.

## Common Pitfalls (Avoid These)

1. ❌ "I'll learn all of ML before I practice mocks"
   - ✅ Instead: Build as you learn

2. ❌ "I'll just read papers all week"
   - ✅ Instead: Read 1-2, code immediately

3. ❌ "I'm waiting for the perfect project idea"
   - ✅ Instead: Ship something simple this week

4. ❌ "Mock scores don't matter, real interviews do"
   - ✅ Instead: Mock scores predict real performance
